# Bundled fonts

## CJKFallback.ttf

Subset of [WenQuanYi Micro Hei](http://wenq.org/wqy2/) containing Latin, CJK Unified
Ideographs, Hiragana/Katakana, Hangul, and CJK punctuation. Shipped so the native PDF
renderer can display CJK text on systems without suitable system fonts (e.g. GitHub
Actions runners and recent macOS releases where PingFang paths are unavailable).

WenQuanYi Micro Hei is licensed under the GPL v3 with font embedding exception. See
https://github.com/anthonyfok/fonts-wqy-microhei/blob/master/LICENSE.txt
